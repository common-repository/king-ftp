== Description ==
 
FTP management is a preliminary feature inside the selection of WordPress images
You can easily connect your downloaded wordpress to WordPress and use the WordPress Media Editor to view and download files in your Web site.
Advantages of king_ftp:
1. Upload multiple files
2. Unlimited FTP connection
3. Delete, display, show link, download
 
== Installation ==

 
1. Upload `king_ftp` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use in media uploader.
